# React Native 0.76 → 0.85.3 Migration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rebuild the SquadGo mobile app on a fresh React Native 0.85.3 project, preserving app identity (`com.squadgoo.au` + upload keystore) and all `src/` code, with vetted-latest dependencies on the New Architecture.

**Architecture:** Generate a clean 0.85.3 scaffold (correct core pins, new-arch defaults), verify it builds *before* adding anything, then layer dependencies in risk order (safe batch → risky/replaced libs one at a time, each verified by a build), copy `src/` + assets + JS/native config, and smoke-test both platforms. Fresh git repo; old 0.76 repo is the rollback.

**Tech Stack:** React Native 0.85.3, React 19.2, New Architecture + Hermes, Reanimated 4 + react-native-worklets, @stripe/stripe-react-native.

**Spec:** `docs/superpowers/specs/2026-06-08-rn-0.85-migration-design.md`

## Conventions for this plan (READ FIRST)

- **Paths:** old project = `/Users/qadirali/Documents/Projects/upwork/PusparajGiri/SquadGo/mobile` (call it `$OLD`). New project is generated in a temp dir then becomes the working project (call it `$NEW`).
- **Identity (must not change):** Android `applicationId`/`namespace` `com.squadgoo.au`; iOS bundle id same as old; reuse the existing upload keystore.
- **Verify-before-proceed:** every dependency phase ends with a **build** (the verification — there are no unit tests for a scaffold migration). If a build fails, STOP and fix before continuing — that's how we avoid whack-a-mole.
- **Commits:** commit per task **in the new repo** (it's the migration's own history + per-phase rollback). Do NOT push until the user has reviewed (per their workflow).
- **8 GB RAM:** close other apps during builds; one platform at a time.
- **Core pins are sacred:** never `yarn add react` / `react-native` / `@react-native/*` — they come from the scaffold.
